master_library.cfg

This is a typical entry in the master_library.cfg file.
As this file cannot be changed via eScript programmes, this is a semi-permanent file. It can be edited using a text editor. That is the only way to remove a book or books from the file. The file can be deleted in the event you want to recreate your Master Library with all new books. This file is located in the /pkg/items/sysbook/config directory.

Please note: Unless you understand the format of config files you should not try deleting individual books. The best option is to delete the file and start with a fresh empty file and add books using the addsysbook command.
