Initial setup for the sysbook package

The sysbook package does not require any setup if you want to build your own Master Library yourself. However, I have included a config file with a library of books from the official UO servers. I am certain I did not find all of the books from the official server but I did get 40 of the "regular" books and the six "Dragonfish" series books that are sold by specialized NPCs on the OSI servers. The pre-made library is named Distro_library.cfg and is in the config directory in the sysbook package. You can rename it to master_library.cfg and use it as your Master Library. If you use the file, you can add your own books to it using the addtomasterlibrary command.
