Sysbook Documentation

For the sysbook package in the Distro, by Yukiko.

Seer Commands

bookro
 Use this command to set a book as read only. Usage is .bookro
You will receive a target to target the book you wish to make read only.
setbook
 This command allows you to change a book's graphic and its read/write status.
Usage is .setbook
You will receive a target. Target a book. A gump appear which allows you to select the book graphic and its read/write status.
setpages
 This command allows the user to modify the maximum number of pages in one or more books. Please note: Due to the unique way books work in UO the maximum number of pages will always be an odd number or else the last page, on the right hand side will be blank. If the user enters 48 as the maximum pages this command will add 1 to that number to make it 49.
Usage is .setpages <page count>
You will be given a target. Target each book one at a time. When you are done setting the maximum pages for books, press <ESC> to cancel the command.
