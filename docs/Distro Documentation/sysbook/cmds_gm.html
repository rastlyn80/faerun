Sysbook package Documentation

For the sysbook package in the Distro, by Yukiko.

Game Master Commands

addspawnshelf
 Allows a user to add one or more bookshelves to the spawn shelf data file. If the user appends a <space> followed by an integer, the integer will be used to set the maximum books that will be spawned in the bookshelves targetted. If no integer is appended to the command, a gump will be presented with the default value stored in the sysbook settings file. If you do not specify an integer this gump will appear for each shelf as you target it. This gives you the opportunity to edit the maximum books on a per shelf basis but having the gump opened for each shelf can be a bit wearysome.
Usage is .addspawnshelf
findbook
 This command presents the user with a gump listing all of the books in the Shard Library. Select the books you wish to create and then they will be created in the container of your choice.
Usage is .findbook
removespawnshelf
 This command allows the user to remove one or more bookshevesf from the spawnedbookshelves data file.
Usage is .removespawnshelf
