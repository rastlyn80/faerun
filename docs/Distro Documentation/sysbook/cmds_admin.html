Sysbook package Documentation - Admin Commands

For the sysbook package in the Distro, by Yukiko.

Administrator Commands

This is a list of the administrator (Admin) level commands and their description:
addtomasterlibrary
 Used to add one or more new books to the Master Library (sysbook:master_library.cfg). After entering the command the user is promted to target a book. The book is checked to be sure it has content and if so, the contents are reformatted to match the books config file format. Then the book info is appended to the end of the Master Library (:sysbook:master_library.cfg). When the user is done, press <ESC> to exit.
Usage is .addtomasterlibrary
addtoshardlibrary
 This command will present a list of books contained in the Master Library. Select the books you wish to add to the Shard Library.  This will make that book available for creation in-game and, if the book is set to spawn when it is selected, it will spawn in bookshelves that are set to spawn books. Note that all of the books presented are set to spawn by default. When adding books you may have special books that you do not want generally available in spawn shelves. These might be books for special quests or events. Unckeck  the spawn check box next to the book to prevent it from spawning in the spawn bookshelves.
Usage is .addtoshardlibrary
createshardlibrary
 This command creates all of the Shard Library's books in a container of your choice. Please ensure the container can hold a lot of items.
Usage is .createshardlibrary
removefromshardlibrary
 This command will remove selected books from the Shard Library. The book will still be available in the Master Library and can be added back into the Shard Library by using the addtoshardlibrary command.
Usage is .removefromshardlibrary
setbooktospawn
 This allows the user to set books in the Shard Library to spawn that have previously been set not set to spawn in the bookshelves.
Usage is .setbooktospawn
setbooktonotspawn
 This command allows the user to stop certain books in the Shard Library from being spawned in the spawn bookshelves.
Usage is .setnottospawn
