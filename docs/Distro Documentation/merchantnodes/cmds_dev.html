Merchant Node - Developer Commands

For the merchantnodes package in the Distro, by Yukiko.

Developer Commands

convertbackupfiles The packed nodes files created by the old merchant node system are not compatible with the new merchant node system. If you have packed nodes files , use this command to convert the files for use with the new merchant node system. The packed node files would be named packedNodes-for-<facet name>. Where <facet name> would be replaced by the name of the facet the nodes were grabbed from, Example: packedNodes-for-Felucca.
Usage is: .convertbackupfiles
disablenodes This will disable all merchant nodes in the selected facet.
Usage is: .disablenodes
enablenodes This will enable all merchant nodes in the selected facet.
Usage is: enablenodes
grabnodes This command is very useful in the event you need to perform an item wipe. It is used to save all nodes, both merchant and guard nodes, and merchant and guard data from those nodes located in the facet you select. To save the nodes in a facet, the user selects it from a gump. The node data is saved to an XML file in /pkg/systems/merchantnodes/backups. All facets are stored in one XML file named nodes.xml.
Usage is .grabnodes
seednodes This is the opposite of grabnodes used to restore grabbed nodes from a file created by the .grabnodes command. The nodes will be restored to the facet the user selects from a gump.
Usage is .seednodes.
