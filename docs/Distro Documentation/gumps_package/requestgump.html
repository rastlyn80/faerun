Constants for RequestGump()

RequestGump
Parameters
(mobile, question, option := "", pre_val := "", cancel_btn:=CANCEL_BTN_OFF)
Name Type
mobile Mobile reference to send the gump to.
question String - Primary text field, instructing the mobile on what to enter (is word wrapped).
option String - Optional text line displayed directly above the input box.
pre_val String - Default text to display in the input box.
cancel_btn CANCEL_BTN_OFF - No cancel button will be shown
CANCEL_BTN_ON - A cancel button will be shown
Explanation
Template to create a single field input gump. See the ".requestgump" command for an example.
Return Values
Returns 'error' if the cancel button is clicked.
Returns 0 if no text entered or the text that was entered.
