Constants for RequestGump()
CONST BTN_TYPE_RADIO          := 0x1;    // Allows selecting one item in the list. "OKAY" must be clicked to return selections.
CONST BTN_TYPE_CHECKBOX    := 0x2;    // Allows multiple selections in the list. "OKAY" must be clicked to return the selections.
CONST BTN_TYPE_NORMAL       := 0x4;    // Allows selecting on item in the list. Gump returns choice when selection is made.
CONST BTN_NOTYPE                 := 0x8;    // Displays a list of strings contained in options with no selection button.

CONST OKAY_BTN        := 0xA00;
 

SelectionGump
Parameters
( mobile, title, options, btn_type:=BTN_TYPE_NORMAL )
Name Type
mobile Mobile reference to send the gump to.
title String that will be used for gump's title.
options Array containing strings to display next to the buttons.
btn_type Constant - Type of button to display next to the items in the list.
Explanation
Creates gump with a list in a gump with optional ways to select an item or
multiple items from the list. See the comments for the button types above.
Return Values
Returns a single value from the options array if RADIO or NORMAL are used.
Returns an array of option values if CHECKBOX is used.
Returns an error, "No option was selected.", if the gump is right clicked .
 

SelectionGump2
Parameters
mobile, title, options, btn_type:=BTN_TYPE_NORMAL, transparent := 0, tooltips :={} )
Name Type
mobile Mobile reference to send the gump to.
title String that will be used for gump's title.
options Array containing strings to display next to the buttons.
btn_type Constant - Type of button to display next to the items in the list.
transparent Integer - If set to 1 makes the gump transparent. This parameter is optional and
defaults to no transparency but if you pass a tooltips parameter you must
specify this parameter.
tooltips Array of tooltips corresponding to each item in the options array.
These can be custom string tooltips or integer cliloc numbers.
This parameter is optional and defaults to an empty array
(no tooltips) if no tooltip array is passed to the function.
Explanation
Creates gump with a list in a gump with optional ways to select an item or
multiple items from the list. See the comments for the button types above.
The gump can be set to transparent.
Tooltips can be assigned to each item in the list and will be displayed when hovering over the item.
Return Values
The user's selection which will be a single string if BTN_TYPE_NORMAL or BTN_TYPE_RADIO
buttons are used. An array of strings selected if BTN_TYPE_CHECKBOX is used.
Error if no selection is made and the gump is closed by right-clicking it.
