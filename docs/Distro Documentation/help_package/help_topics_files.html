In the gump, the help topics are displayed within an HTML element. When you edit or create the topic files, use HTML tags to format them properly.


The Help Topics List
In order to display a help topic in the help gump it must be added to helptopics.cfg located in /systems/help/config. In this file you enter the topic title and associated TXT file containing the HTML formatted text to be displayed.
This is what a typical helptopics.cfg looks like:


# $Id: helptopics.cfg 614 2006-07-06 15:23:38Z austinheilman $
#
#

Topic    [Shard] Behavior
{
    File    behavior
}
Topic    [Shard] Credits
{
    File    credits
}
Topic    [Shard] Rules
{
    File    rules
}
Topic    [System] Awards
{
    File    awards
}
Topic    [System] Commands
{
    File    commands
}
Topic   [System] Help
{
    File    help
}
Topic   [System] Skills
{
    File    skills
}


The topic entry for Credits is:
Topic    [Shard] Credits       ---> The text following the keyword Topic will appear in the help topics gump.
{
    File    credits                    ---> This is the name of the TXT file that will be read from the helptopics directory.
}
