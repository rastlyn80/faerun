Initial setup for the help package

The help package will require some setup if you wish to customize it for your server. You should familiarize yourself with the file helptopics.cfg in /systems/help/config and the various help topics files found in /systems/help/helptopics. To learn more about these files. See Help Topics Files for instructions on how to set-up topics for the help system.
