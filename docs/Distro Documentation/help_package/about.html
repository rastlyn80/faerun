Help package Documentation

For the help package in the Distro, by Yukiko.

About the help package:

The Help package is activated when a player clicks the help button on their paper doll or when the .help command is used. This documentation explains how to customize the package as well as its functionality, most from a staff member's perspective. The package allows the shard developer to setup topics which a player can view. It also has a self help feature for players who might get stuck and are unable to move. The player can page a staff member in the event they need help that might require staff intervention.
