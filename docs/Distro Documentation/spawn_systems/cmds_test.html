Merchant and Guard Node - Admin Commands

For the merchantnodes package in the Distro, by Yukiko.

Administrator Commands

This is a list of the administrator (Admin) level commands and their description:
forceregionspawn
 This command force a region to spawn when used, regardless of when it is scheduled to spawn. Usage is .forceregionspawn <region name.
restartspawner
 This command does what the name says, restarts the spawner.
